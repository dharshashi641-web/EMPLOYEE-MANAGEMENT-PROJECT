🧑‍💼 Employee Management System
📌 Project Description

This is a console-based Employee Management System built using Python. It helps in managing employee records, attendance, departments, and generating simple reports using JSON file storage.

🚀 Features
👤 User Registration and Login system
🧑‍💼 Add new employees (auto ID generation)
📋 View all employee details
✏️ Update employee information
❌ Delete employee records
📁 Data stored using JSON files
🛠️ Technologies Used
Python 🐍
File Handling (JSON)
OOP Concepts (if used)
